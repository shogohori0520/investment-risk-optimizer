付録A: 重要公式集

基本公式

複利（将来価値）

FV = PV × (1 + r)n

PV=現在価値, r=年利, n=年数。例: 100万円, 5%, 10年 → 100万×1.0510 = 162.9万円

年金終価（積立の将来価値）

FV = PMT × [(1 + r)n − 1] / r

PMT=毎期の積立額。例: 月3万円, 年5%(月0.417%), 30年(360ヶ月) → 約2,497万円

年金現価

PV = PMT × [1 − (1 + r)−n] / r

72の法則

資産が2倍になる年数 ≒ 72 ÷ 利率(%)

バリュエーション公式

PER / PBR / ROE

PER = 株価 / EPS　　PBR = 株価 / BPS　　ROE = 純利益 / 自己資本

デュポン分解

ROE = (純利益/売上) × (売上/総資産) × (総資産/自己資本)

CAPM

E(Ri) = Rf + β × [E(Rm) − Rf]

WACC

WACC = E/(E+D) × Re + D/(E+D) × Rd × (1−T)

DCF（企業価値）

V = Σ FCFt / (1+WACC)t + TV / (1+WACC)n

ゴードン成長モデル

P = D1 / (r − g)

D1=来期配当, r=要求収益率, g=配当成長率。例: D1=100円, r=8%, g=3% → P=2,000円

リスク・パフォーマンス公式

シャープレシオ

SR = (Rp − Rf) / σp

ソルティノレシオ

SoR = (Rp − Rf) / σdownside

VaR（パラメトリック法）

VaR = μ − z × σ

z=信頼水準に対応するz値（95%→1.645, 99%→2.326）

ポートフォリオ分散（2資産）

σp² = w1²σ1² + w2²σ2² + 2w1w2σ1σ2ρ12

テクニカル指標公式

RSI

RSI = 100 − 100 / (1 + RS)　　RS = 平均上昇幅 / 平均下落幅

ボリンジャーバンド

上限 = SMA + k × σ　　下限 = SMA − k × σ

MACD

MACD = EMA(12) − EMA(26)　　Signal = EMA(9) of MACD
